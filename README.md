# YourVue

学习 vue 源码的最好方法，就是自己实现一个 vue !

本项目旨在实现 vue 的主流程，理解 vue 主流程中的细节，省去额外的代码方便查看与学习。

- [x] new YourVue
- [x] compiler
- [x] observer
- [x] vdom
- [x] component
- [x] lifecycle
- [x] next-tick
- [ ] watch computed
- [ ] event
- [ ] slot
- [ ] v-if v-for ...
- [ ] vue-router
- [ ] vuex

## Project setup
```
yarn install
```

yarn serve/dev
