# YourVue

学习 vue 源码的最好方法，就是自己实现一个 vue !

本项目旨在实现 vue 的主流程，理解 vue 主流程中的细节，省去额外的代码方便查看与学习。

- [x] new YourVue
- [x] compiler
- [x] observer
- [x] vdom
- [x] component
- [x] lifecycle
- [x] next-tick
- [x] props (parent to child)
- [x] v-if v-for
- [x] watch computed
- [x] event (dom & eventEmitter)
- [x] slot (普通插槽)

实现中...
- [ ] keep-alive
