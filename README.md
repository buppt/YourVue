# YourVue

学习 vue 源码的最好方法，就是自己实现一个 vue !

本项目旨在实现 vue 的主流程，理解 vue 主流程中的细节，省去额外的代码方便查看与学习。

最新的代码在`src`中，各功能的实现过程在`oldSrc`中，可以通过改变`vue.config.js`文件来改变`main.js`的入口进行调试。
